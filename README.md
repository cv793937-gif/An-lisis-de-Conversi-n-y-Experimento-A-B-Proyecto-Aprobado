Análisis de Conversión y Experimento A/B — Proyecto Aprobado
📌 Descripción del Proyecto
Este proyecto analiza el desempeño de dos landing pages (A y B) mediante un experimento A/B, evaluando su impacto en la tasa de conversión y el gasto promedio.
El objetivo es identificar cuál página genera mejores resultados y qué variables influyen en la conversión.

El análisis incluye:

Exploración de datos

Pruebas estadísticas (z-test)

Visualización de variables categóricas

Interpretación de resultados

Recomendaciones de negocio

📈 Resultados Clave
✔️ 1. Página con mayor conversión
La landing B mostró una tasa de conversión significativamente mayor.

✔️ 2. Gasto promedio
Los usuarios que convirtieron en la página B también mostraron un gasto promedio superior.

✔️ 3. Variables categóricas
La fuente de tráfico no mostró diferencias significativas.

El tipo de usuario tampoco influyó en la conversión.

💡 Recomendaciones de Negocio (priorizadas por impacto en “converted”)
Implementar la landing page ganadora (B) para maximizar conversiones y revenue.

Redirigir más tráfico hacia la página B, especialmente desde campañas pagadas.

🏁 Conclusión
El experimento A/B demuestra que la landing B es la opción óptima para incrementar conversiones y mejorar el rendimiento del funnel.
Este análisis sirve como base para futuras optimizaciones y experimentos adicionales.
